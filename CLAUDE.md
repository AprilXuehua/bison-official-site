# BISON Official Site 프로젝트

## 프로젝트 개요
- BISON 회사 소개 랜딩페이지
- Next.js + Tailwind CSS
- 배포: https://bison-official-site.vercel.app/

## 디자인 원칙
- 피그마 디자인 기준: Bison Official Site v0 > 메인페이지_Black
- 반응형 웹 필수 (모바일/태블릿/데스크탑)

### Header(GNB)
- 데스크탑: 1920 x 75px
- 배경: 글라스모피즘 (blur 처리)
- 고정 여부: sticky top-0

### Footer
- 데스크탑: 1920 x 325px

### 반응형 브레이크포인트
- 모바일: 375px
- 태블릿: 768px
- 데스크탑: 1920px

## 코딩 규칙
- 파일 수정/생성 전 반드시 나한테 먼저 물어볼 것
- 컴포넌트는 src/components 폴더에 저장
- 한국어로 소통할 것

## 하지 말 것
- package.json 임의 수정 금지
- .env 파일 절대 수정 금지
